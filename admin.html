Deletion Troubleshooting

If records aren't being removed, it's likely due to Row Level Security (RLS) in Supabase. By default, anonymous users cannot delete data.

How to fix:

  1. Go to your Supabase Dashboard.
  2. Navigate to Table Editor > select contacts table.
  3. Click RLS Disabled (or Add Policy).
  4. Create a new policy using "Enable Delete for Anon" template or use this SQL in the SQL Editor:
CREATE POLICY "Enable delete for anonymous users" 
ON public.contacts 
FOR DELETE 
USING (true);
